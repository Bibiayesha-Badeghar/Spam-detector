ML Portfolio Project

Email Spam Detector

A full-stack machine learning app that classifies messages as spam or legitimate. Built with Python, scikit-learn, and Flask — trained on 5,600+ real messages with ~97.5% accuracy.

Check a Message
System Workflow
1
Ingest Data Paste an email or SMS-style message into the form.
2
ML Analysis A Random Forest classifier scores the text using TF-IDF features from the training dataset.
3
Feedback Loop Correct uncertain predictions to improve the model via optional retraining.
