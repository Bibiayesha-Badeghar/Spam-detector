# 🚀 Spam Detector with Hardware Monitoring
 
A beginner project to learn how AI models use your computer's CPU, RAM, and disk.
 
## 🎯 What This Does
 
- Trains a spam detection model
- Saves and loads the model
- Makes predictions on new emails
- Monitors CPU, RAM, and disk usage
 
## 🛠 Installation
 
```bash
pip install scikit-learn psutil
```
 
## ▶️ How to Run
 
```bash
python spam_detector_project.py
```
 
## 📊 What You'll See
 
- CPU spikes during training (50-100%)
- CPU drops during predictions (<5%)
- Real-time hardware monitoring
 
## 🎓 What You'll Learn
 
✅ How to train a machine learning model  
✅ How to save/load models from disk  
✅ How AI uses your computer's hardware  
✅ Why training is expensive but predictions are cheap
